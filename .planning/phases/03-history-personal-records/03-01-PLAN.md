**1. `computePRBestsByClient(supabase, clientId)`**
- Queries ALL completed set_logs (completed=true) for a client across all their completed workout sessions

- Query pattern:
  ```typescript
  const { data } = await supabase
    .from('set_logs')
    .select(`
      exercise_id,
      weight_kg,
      reps,
      completed,
      workout_session:workout_sessions!set_logs_session_id_fkey (
        client_id,
        completed
      )
    `)
    .eq('completed', true)
  ```
  Then filter to only rows where `workout_session.client_id === clientId && workout_session.completed === true`.
  Build the Map by iterating and updating max volume per exercise_id.

- Cast the result: `const rows = (data ?? []) as Array<{ exercise_id: string; weight_kg: number; reps: number; workout_session: { client_id: string; completed: boolean } | null }>`

**2. `detectSessionPRs(supabase, clientId, sessionId, exerciseIds)`**
- Purpose: for a specific session, determine which exercises achieved a PR compared to ALL OTHER completed sessions (excluding the current session)
- Returns `Set<exerciseId>` of exercises that set a PR in this session
- Two-step approach:
  a. For each exerciseId, find the best volume from the current session's completed set_logs
  b. Find the best volume from all OTHER sessions (completed=true, session_id !== sessionId)
  c. If current session best > historical best (or historical best is 0), include in the returned Set
- Query for current session best:
  ```typescript
  const { data: currentLogs } = await supabase
    .from('set_logs')
    .select('exercise_id, weight_kg, reps')
    .eq('session_id', sessionId)
    .eq('completed', true)
    .in('exercise_id', exerciseIds)
  ```
- Query for prior sessions best: use `computePRBestsByClient` logic but exclude the current session by querying set_logs where session_id != sessionId. You can accomplish this by:
  ```typescript
  const { data: priorLogs } = await supabase
    .from('set_logs')
    .select(`
      exercise_id, weight_kg, reps,
      workout_session:workout_sessions!set_logs_session_id_fkey (client_id, completed)
    `)
    .eq('completed', true)
    .neq('session_id', sessionId)
    .in('exercise_id', exerciseIds)
  ```
  Filter to only rows where `workout_session.client_id === clientId && workout_session.completed === true`.

Types: Use `import type { SupabaseClient } from '@supabase/supabase-js'` for the supabase parameter type. Do NOT use `any`.

2. For each session, compute `hasPR` by importing `detectSessionPRs` from `@/lib/pr-detection`:
   ```typescript
   import { detectSessionPRs } from '@/lib/pr-detection'
   ```

   Since calling `detectSessionPRs` per session would issue N queries, batch it differently — instead compute a single "all-time best" map ONCE (for all exercises of this client) and then compare per-session:

   Import `computePRBestsByClient` instead. For each session, find the best set in that session and compare against the global map. Mark `hasPR = true` if ANY exercise in this session has a best `weight_kg * reps` that equals the global best (meaning it was achieved in this session — but careful: the global best already includes this session).

   Actually the correct approach for history feed is: a session "has a PR" means it was the FIRST time (or best time) the client hit a new all-time best for at least one exercise. This is complex to compute retroactively without per-session exclusion.

   **Simpler practical approach:** call `detectSessionPRs` in a `Promise.all` for all sessions. The total queries = 2 per session (current + prior). For the history page, batch it: use `computePRBestsByClient` once to get global bests, then for each session compute its own best per exercise and see if the session's best matches the global best AND the global best was achieved in that session.

   Use this approach:
   ```typescript
   import { computePRBestsByClient } from '@/lib/pr-detection'

   // After fetching sessions:
   const globalBests = await computePRBestsByClient(supabase, client.id)

   const sessions: SessionData[] = sessionsFromDb.map((session) => {
     const completedLogs = (session.set_logs ?? []).filter((log) => log.completed)
     const totalVolume = completedLogs.reduce((sum, log) => sum + log.weight_kg * log.reps, 0)
     const trainedMuscles = Array.from(new Set(
       completedLogs.map(log => log.exercises?.muscle_group?.trim()).filter((m): m is string => Boolean(m))
     ))

     // A session "hasPR" if any completed set in it matches the global best for its exercise
     const hasPR = completedLogs.some((log) => {
       if (!log.exercises?.id) return false
       const globalBest = globalBests.get(log.exercises.id) ?? 0
       const setVolume = log.weight_kg * log.reps
       return globalBest > 0 && setVolume >= globalBest
     })

     return {
       id: session.id,
       started_at: session.started_at,
       finished_at: session.finished_at,
       workout_day: session.workout_day,
       durationLabel: formatDuration(session.started_at, session.finished_at),
       totalVolume,
       completedSets: completedLogs.length,
       trainedMuscles,
       hasPR,
     }
   })
   ```

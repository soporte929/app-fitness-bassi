**Step 2 — Remove `isPR` from `ExerciseCard` props destructure and compute it from sets state:**

The current code renders `{exercise.isPR && <Trophy badge>}` using the static prop. Change this to compute `isPR` dynamically.

Inside `ExerciseCard`, after the `sets` state declaration, add a derived value:

```typescript
const isPR = exercise.prBestVolume !== undefined
  ? sets.some((s) => {
      if (!s.done) return false
      const vol = (parseFloat(s.weight) || 0) * (parseInt(s.reps) || 0)
      return vol > 0 && vol > exercise.prBestVolume!
    })
  : (exercise.isPR ?? false)  // fallback to static prop if prBestVolume not provided
```

This means:
- If `prBestVolume` is provided: `isPR` is true when ANY completed set in the current render exceeds the prior best
- If `prBestVolume` is not provided: fall back to the static `isPR` prop (backward compat for any callers that don't pass prBestVolume)

# Plan 12.1: Data Layer Migration & Goal Visualization

## Objective
Migrate the `/progress` page data fetching from legacy tables to the new unified `client_measurements` table, create the logging server action, and add a visual reference line for the target weight on the chart.

## Context
- `.planning/ROADMAP.md` (Phase 12)
- `app/(client)/progress/page.tsx`
- `components/client/progress-charts.tsx`
- `lib/supabase/types.ts` (`client_measurements` structure)

## Tasks

<task type="auto">
  <name>Create Server Action</name>
  <files>
    - `app/(client)/progress/actions.ts` (create)
  </files>
  <action>
    - Create a new file for progress server actions.
    - Export `logClientMeasurementAction(clientId: string, payload: { weight_kg?: number, body_fat_pct?: number, waist_cm?: number, hip_cm?: number, chest_cm?: number, arm_cm?: number, thigh_cm?: number })`.

  </action>

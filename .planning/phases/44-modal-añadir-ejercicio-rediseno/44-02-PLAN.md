# Phase 44 — Gap Closure: Fix ExercisePicker Modal Height

## Background
Durante las pruebas de la Phase 44, se ha reportado un bug visual en el componente `ExercisePicker`: en dispositivos móviles (aprox. 390px de ancho) el modal aparece cortado por la parte superior. El título "Añadir ejercicio" y el botón de cerrar (X) no son visibles debido a que la caja del modal excede la altura disponible en el viewport, ocultando el header.

## Goal
Asegurar que el header del modal `ExercisePicker` sea siempre visible en dispositivos móviles, redimensionando/limitando su altura máxima de forma dinámica para evitar que se desborde el viewport verticalmente.

## Tasks

- [ ] **1. Ajustar altura máxima en `ExercisePicker`**
  - **Archivo**: `components/trainer/exercise-picker.tsx`
  - **Acción**: Modificar las clases de Tailwind del contenedor del modal (la caja interior con `max-h-[85vh]`).
  - **Detalle**: Reemplazar `max-h-[85vh]` por una medida más ajustada o responsiva, por ejemplo `max-h-[75dvh] min-[431px]:max-h-[85vh]`. Usar unidades `dvh` donde sea posible es ideal en mobile para prevenir el problema de la barra de navegación variable. Garantizar que solo los elementos interiores (la lista de ejercicios `flex-1 overflow-y-auto`) hagan scroll, y que el header y el input se queden visibles independientemente de la resolución y teclado.
  - **Resultado Esperado**: El modal en mobile no supera el viewport, y su cabecera se mantiene siempre en pantalla.

- [ ] **2. Ejecutar fix y validar**
  - Aplicar el cambio usando las tools del sistema.
  - Si es posible arrancar la app on local, verificar visualmente/técnicamente.
  - Reportar cualquier advertencia de renderizado.

## Definition of Done
- El código se ha modificado limitando la altura vertical del `ExercisePicker` a `75dvh` (o valor equivalente sugerido) en mobile.
- El header de la vista en mobile (`<390px` o de ancho) donde aparece "Añadir ejercicio" se renderiza completamente dentro de los límites del viewport.
- Escalar/cerrar el modal funciona como se espera sin ser interrumpido.

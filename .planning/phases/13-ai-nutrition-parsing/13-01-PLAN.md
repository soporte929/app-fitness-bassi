Export `parseNutritionAction(rawDescription: string): Promise<{ success: true; data: MacroEstimate } | { success: false; error: string }>`:

1. Guard: if `!process.env.ANTHROPIC_API_KEY` return `{ success: false, error: 'Servicio de IA no configurado' }`
2. Guard: if `!rawDescription.trim()` return `{ success: false, error: 'Descripción vacía' }`
3. Instantiate: `const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY })`
4. Call `anthropic.messages.create()` with:
   - model: `claude-sonnet-4-20250514` (per CLAUDE.md)
   - max_tokens: 256
   - Single user message: prompt that asks for Spanish nutrition expert estimation, instructs JSON-only response with exact keys `kcal`, `protein_g`, `carbs_g`, `fat_g`, `description`, and instructs `{"error":"no_parse"}` if ininteligible. Embed `rawDescription.trim()` in the prompt.
5. Extract text block: `message.content.find(b => b.type === 'text')`; if none return `{ success: false, error: 'Respuesta vacía de Claude' }`
6. Strip markdown fences before JSON.parse: `.replace(/^```json?\s*/i, '').replace(/\s*```$/, '')`
7. If `parsed.error === 'no_parse'` return `{ success: false, error: 'No se pudo interpretar la descripción' }`
8. Coerce all numeric fields with `Number()` — never cast directly (Claude may return strings). Round kcal with `Math.round`, apply `.toFixed(1)` for protein/carbs/fat then wrap in `Number()`.
9. Return `{ success: true, data: MacroEstimate }`.
10. Catch block: return `{ success: false, error: err instanceof Error ? err.message : 'Error desconocido' }`.

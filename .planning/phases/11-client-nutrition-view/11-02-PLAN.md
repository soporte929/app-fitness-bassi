# Plan 11.2: Client Meals List & Equivalent Swapping

## Objective
Build the daily planned meals section for the client's nutrition view. Display the structured groups (Meal 1, Meal 2, etc.) for Diet Type A/B based on `nutrition_plan_meals` and `meal_plan_items`. Implement the UI to swap equivalents for Diet Type B and a function to log the meal (moving it to `food_log`).

## Context
- `.planning/SPEC.md` (or ROADMAP.md Phase 11)
- `lib/supabase/types.ts`
- `app/(client)/nutrition/page.tsx`
